# Gemini用：面談メモデモデータ生成プロンプト

以下のプロンプトをGeminiに入力して、面談メモのデモデータを生成してください。

---

## プロンプト

```
あなたは化学メーカーの営業担当者です。顧客との面談メモを作成してください。

【要件】
- 化学メーカーの研究開発部門向けの面談メモとして、具体的で実用的な内容を作成すること
- 以下のセクションを含めること：
  1. 現状の課題（現在使用している材料や技術の問題点）
  2. 具体的なニーズ（数値や仕様を含む）
  3. 用途（どのような製品・用途に使用するか）
  4. その他の要望（コスト、納期、環境対応など）

【必須の技術要件（数値を含めること）】
以下のいずれかを含めること：
- 温度条件（例: 100℃以上、-20℃から80℃まで、-40℃以下など）
- 強度・物性（例: 引張強度300MPa以上、弾性率70GPa以上、圧縮強度0.5MPa以上など）
- 耐性（例: 耐熱性120℃以上、耐薬品性、耐候性10年以上など）
- その他の具体的な数値や仕様（厚み、透過率、抵抗率、耐久年数など）

【業界・用途の例】
- 自動車部品（ボディパネル、内装材、タイヤなど）
- 電子機器（電池、ディスプレイ、半導体関連など）
- 航空機・鉄道（内装材、防振材など）
- 化学プラント（配管、容器など）
- 建築・インフラ（外装材、構造材など）
- 医療・食品（包装材、容器など）

【出力形式】
- 日本語で自然な面談メモの形式で記述
- 箇条書きや見出し（【】）を使用して構造化
- 企業名や部署名は具体的に記載（例: サンプル自動車株式会社、電池開発部など）
- 数値は必ず含めること（単位も明記）
- 500文字以上1000文字程度の分量

【例】
以下のような形式で作成してください：

「[材料名/技術名]について相談がありました。

【現状の課題】
- 現在使用している[材料名]は、[具体的な問題点]
- [追加の問題点]

【具体的なニーズ】
- [物性名]: [数値]以上が必要
- [物性名]: [数値]以上
- 使用温度範囲: [数値]から[数値]まで
- [その他の要件]: [数値や仕様]

【用途】
- [具体的な用途]
- [使用規模や数量]

【その他の要望】
- [コスト、納期、環境対応などの要望]」

上記の要件に従って、化学メーカー向けの面談メモを1件作成してください。
```

---

## 使い方

1. 上記のプロンプトをGeminiにコピー＆ペースト
2. 生成された面談メモをコピー
3. R&D Brainアプリの「面談メモ (Raw Content)」欄に貼り付けて使用

## バリエーション生成のコツ

複数のデモデータを作成したい場合は、プロンプトの最後に以下を追加してください：

- 「[業界名]向けの面談メモを作成してください」（例: 自動車向け、電子機器向け）
- 「[材料タイプ]に関する面談メモを作成してください」（例: 樹脂材料、ゴム材料、金属材料）
- 「[用途]向けの面談メモを作成してください」（例: 軽量化、難燃性、耐薬品性）

## 注意事項

- 生成された面談メモには必ず具体的な数値が含まれていることを確認してください
- 数値がないと、AIレビューで「情報不足」と判定される可能性があります
- 必要に応じて、生成後に数値や仕様を追加・修正してください
